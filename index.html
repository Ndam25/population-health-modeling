Analytical Pipelines

R · Pipeline 1

Multi-Country DHS Harmonization

Automated QA/QC (range checks, outlier flagging, missingness audit), complex survey design specification, population-weighted estimates via the survey package. Full audit trail logged.

# Complex survey design svy_design <- svydesign( ids = ~cluster_id, strata = ~stratum, weights = ~weight, data = dhs_clean )
R · Pipeline 2

PAF Burden Estimation (CRA)

Population-Attributable Fractions of malnutrition on HIV/TB/diarrheal disease. GBD-adapted Comparative Risk Assessment framework. Bootstrap CIs (n=500) + Probabilistic Sensitivity Analysis (PSA, n=1000).

# Levin's PAF formula levin_paf <- function(or, pe) { (pe * (or - 1)) / (pe * (or - 1) + 1) }
R · Pipeline 3

SEIR-Nutrition Compartmental Model

SEIR extended with nutritional strata. Nutritional status modulates β (transmission) and μ (case-fatality). Three policy scenarios projected over 5 years. Solved via deSolve::ode() LSODA.

# Stunted stratum: 45% higher β beta_stunt = beta_base * 1.45 mu_stunt = 0.018
R · Pipeline 4

ARIMA + Bayesian Forecasting

5-year disease burden forecasting: auto.arima, StructTS state-space, and Bayesian Structural Time Series (bsts) compared. Monte Carlo uncertainty envelope (n=1000 trajectories). Explicit PI communication.

# bsts: local linear trend ss <- AddLocalLinearTrend( list(), y = hiv_series) fit <- bsts(hiv_series, state.specification = ss, niter = 2000, seed = 42)
Python · Pipeline 5

MICE Missing Data Pipeline

Multiple Imputation by Chained Equations (m=20 datasets). MAR diagnostics, BayesianRidge estimator per variable, Rubin's Rules pooling with between-imputation variance quantification. Bias vs. complete-case analysis reported.

# Rubin's Rules pooling T = W + (1 + 1/m) * B # W = within, B = between
Python · Pipeline 6

Spatial Small-Area Estimation

Model-based geostatistics for subnational disease burden mapping. Calibrated against synthetic DHS cluster coordinates. Produces uncertainty-quantified subnational PAF maps for policy targeting.

# SAE: model-based prediction # at unsampled locations KrigingPredictor( kernel='Matern', nu=1.5)

Methods Reference

R

Complex survey estimation

survey package — stratified cluster design, probability weights, design-corrected SEs. Applied to DHS/MICS population inference across 10+ countries.

R

Compartmental modeling (ODE)

deSolve::ode() LSODA solver — SEIR-Nutrition system with 10 differential equations. Calibrated against synthetic IeDEA cohort data (17,000+ patient equivalent).

R

Bayesian structural time-series

bsts — local linear trend + MCMC (2000 iterations). Posterior predictive intervals. Compared vs. ARIMA and StructTS for ensemble averaging.

Py

Multiple Imputation by Chained Equations (MICE)

sklearn.impute.IterativeImputer + BayesianRidge. m=20 datasets. Rubin's Rules pooling. Bias quantification vs. complete-case analysis. MAR diagnostics via missingness correlation matrix.

Py

Probabilistic Sensitivity Analysis

Monte Carlo simulation (n=1000) propagating parameter uncertainty through PAF and forecast models. Tornado diagrams for one-way sensitivity. All uncertainty communicated as explicit distributions.
